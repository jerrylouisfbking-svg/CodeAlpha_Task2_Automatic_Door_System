# Automatic Door Opening System Using Ultrasonic Sensor and Servo Motor

## Description
This project demonstrates a smart automation system using Arduino, HC-SR04 Ultrasonic Sensor, Servo Motor, and LED. The ultrasonic sensor detects nearby objects and automatically opens the door using a servo motor. The LED indicates the door status.

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Applications
- Smart Homes
- Hospitals
- Offices
- Shopping Malls
- Public Buildings

## Software Used
- Tinkercad
- Arduino IDE
